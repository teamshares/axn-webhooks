# axn-webhooks

An [axn](https://github.com/teamshares/axn)-consuming gem.

## Installation

Add to your Gemfile:

```ruby
gem "axn-webhooks"
```

## Development

- `bin/refresh` — pull latest and install dependencies (fails on a dirty working tree).
- `bundle exec rake` — run the default task (specs + rubocop) before pushing.
